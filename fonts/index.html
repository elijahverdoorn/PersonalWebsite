Model ALU

The Arithmatic Logic Unit (ALU) is an integral part of the CPU. In St. Olaf College's CSCI241: Hardware Design, an assingment was given to write an implementation of the processes of the ALU in a high-level programming language. My solution, written in C++, can be viewed on GitHub.

Assembly Language Exploration Project

Assembly languages are low-level programming languages that present unique challenges to programmers. For the final project in CSCI241: Hardware Design, the class was tasked to create a text-analysis and string-substitution program. Written in as88 assembly, my creation can be found by following the link below to my GitHub page!
