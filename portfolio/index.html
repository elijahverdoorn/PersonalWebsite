Model ALU

The Arithmatic Logic Unit (ALU) is an integral part of the CPU. In St. Olaf College's CSCI241: Hardware Design, an assingment was given to write an implementation of the processes of the ALU in a high-level programming language. My solution, written in C++, can be viewed on GitHub.

Assembly languages are low-level programming languages that present unique challenges to programmers. For the final project in CSCI241: Hardware Design, the class was tasked to create a text-analysis and string-substitution program. Written in as88 assembly, my creation can be found by following the link below to my GitHub page!

Operating Systems Kernel

I worked in operating systems class to extend the Linux Kernel. My version of the kernel extends the abilities of the operating system by adding system calls. Adding these system calls taught me about the functions of the operating system and exposed me to the existing code. Also important to the project was the use of Virtual Machines, which were used to allow for easy testing in a safe envionment.

Web-Rendering of 3D Models

In a month-long project, I built a client-server system to render a large 3D model of the interior of a building on the web. Attention was taken to the potential of mobile display, responsiveness, and user experience. Testing was done across platforms, and the results were published in the proceedings of the 2016 Midwestern Instructional Computing Symposium. This project also lead to a speaking engagement sponsored by the Department of Mathematics, Statistics, and Computer Sciences at St. Olaf College in April 2016.

Training Android Developers

In collaboration with the St. Olaf department of Mathematics, Statistcs, and Computer Sciences, I developed a curriculum called Training Android Developers that took computer science students with no knowledge of Java through the basics of the Java language, then through a series of lessons designed to instruct in the basics of creating Android applications. I then used this curriculum to teach CSCI284: Mobile Computing Applications to students in the spring of 2016.

Members

St. Olaf uses WordPress as a content management system, and needed a way to display the members of various organizations. I built a plugin for WordPress to do just that. The requesters were so pleased that they came back to me to ask for the addition of functionality to manage the various office hours that are held. After adding these features, the plugin was deployed and is used in production today.
